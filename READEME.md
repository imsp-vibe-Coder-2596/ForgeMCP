# ForgeMCP

ForgeMCP is an enterprise-grade TypeScript framework for building Model Context Protocol (MCP) servers.

## Vision

Provide a modular, transport-agnostic runtime that allows developers to focus on business logic while the framework handles infrastructure concerns such as dependency injection, configuration, logging, middleware, validation, observability, lifecycle management, and extensibility.
